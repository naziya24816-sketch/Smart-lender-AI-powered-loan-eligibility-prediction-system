# SmartLender – Loan Eligibility Prediction

## Project Overview
SmartLender is a Machine Learning-based web application that predicts whether a loan applicant is eligible for a loan based on financial and personal details. The project helps automate loan approval decisions and improve efficiency.

## Features
- Loan eligibility prediction
- User-friendly interface
- Machine Learning model
- Fast and accurate predictions

## Technologies Used
- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- HTML/CSS

## Project Structure
```
SmartLender/
│── app.py
│── requirements.txt
│── README.md
│── templates/
│── static/
│── model/
```

## Installation
1. Clone the repository.
2. Install the required packages:
```
pip install -r requirements.txt
```
3. Run the application:
```
python app.py
```
